Detailed Description

template<typename OptionType>
struct curlpp::internal::OptionContainerType< OptionType >

Defines types connected with the type of the option.

To be able to operate on option's values we have to define value of what type do we accept as an argument (ParamType), store (ValueType), return (ReturnType).
