Detailed Description

template<typename OptionType>
class curlpp::NotAvailableOptionTrait< OptionType >

This class is used when the option is not implemented.

Constructor & Destructor Documentation

template<typename OptionType >
curlpp::NotAvailableOptionTrait< OptionType >::NotAvailableOptionTrait ( )

The constructor will contain an unset option value.

Note that if you try to retreive the value of this option before calling the curlpp::Option::setValue function it will throw a UnsetOption exception.
